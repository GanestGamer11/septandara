O Jogo

As aventuras de Dandara. Uma viajem pelo reino da matemática

As aventuras de Dandara é um jogo educativo com objetivo de quebrar a barreira chata do aprender usando de uma abordagem de gameficação dos conteúdos de matemática

Esse jogo foi desenvolvido como um Projeto escolar do nosso curso de informática para internet. Criamos esse jogo pensando em uma forma de melhorar a aprendizagem dos conteúdos de matemática de uma forma mais leve e divertida.

Para o desenvolvimento do projeto, utilizamos de algumas ferramentas e linguagens importantes como:

  • Unity 2D
  • C#
  • Photoshop

Todo o projeto foi criado e desenvolvido com apoio de nossa instituição IFRN-Caicó. Criado e pensado desde o começo em ser uma ferramenta gratuita de ensino e reforço do ensino.
